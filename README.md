# ISTQB Test Design Techniques

This repository demonstrates core **black-box test design techniques** based on the ISTQB Foundation Level syllabus.  
Each technique is explained with a practical scenario and structured test cases.

---

## 📌 Covered Test Techniques
- **Boundary Value Analysis (BVA)**  
- **Equivalence Partitioning (EP)**  
- **Decision Table Testing (DT)**  
- **State Transition Testing (STT)**  

---

## 🎯 Purpose
The purpose of this repository is to demonstrate **analytical thinking** and **structured test case design** using industry-recognized testing techniques.  
This shows the ability to:  
- Identify input ranges, partitions, or states  
- Design minimal yet effective test cases  
- Ensure coverage for valid and invalid scenarios
